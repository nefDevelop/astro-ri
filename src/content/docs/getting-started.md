## Installation

Install Ri using npm:

```bash
npm create astro@latest -- --template ri
```

## Configuration

Configure your site in `src/config/palette.js`:

```js
export const PALETTE_SEED = '#FF7800';
export const HARMONY = 'tonal';
```
